# react-typescript-app
# Базовый код React 18 с TypeScript, выводящий "Hello World!"

# Быстрый старт с Create React App
```bash
npx create-react-app my-app --template typescript
```

# Структура проекта:

```text
my-app/
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── App.tsx
│   ├── index.tsx
│   ├── index.css
│   └── ...
├── package.json
├── tsconfig.json
└── ...
```

Для старта
```bash
npm i
npm start
```
